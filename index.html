学習の見方

  1. まず API グループ名を見る
  2. 次に 値 / 由来 / コード例 を対応させる
  3. そのあと DevTools で同じ式を試す
source取得元の系統
value現在の取得値
code実際のコード例

location 系

現在のURLやホスト、パスなど、ページの位置を表す情報です。

document 系

現在のHTML文書や参照元、タイトルなど、ドキュメントに関する情報です。

navigator 系

ブラウザ・端末・言語・Cookie有効状態など、利用者環境の自己申告的な情報です。

screen / window 系

画面サイズや表示領域、ピクセル比など、見た目や表示に関係する情報です。

Intl / Date 系

タイムゾーンや現在時刻など、ロケールや時刻表現に関する情報です。

performance 系

ページ読込やリソース読込のタイミング情報です。性能分析や学習に向いています。

機能有無チェック

localStorage や indexedDB など、「その機能が使えるか」を確認する例です。

取得結果(JSON)

学習用に、現在ページで取得した内容をまとめて JSON 表示しています。
